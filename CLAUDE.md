# 2026 年 習慣管理プロジェクト

このディレクトリは、2026 年に毎日コツコツ取り組む習慣を管理・記録するためのものです。

## 管理する習慣

### 基本習慣

- ラジオ体操
- シダキュア（花粉症の薬）
- スクワット（目標: 20 回）
- 筋トレ（腕立て伏せ、腹筋 各 20 回）
- Typnix 練習（タイピング練習アプリ）

### 学習習慣

- **Sociomedia ヒューマンインターフェース ガイドライン**（略称: **shig**）

  - 100 トピックを 1 日 1 つずつ読んでいく
  - https://www.sociomedia.co.jp/category/shig

- **Documentation for Ruby 4.0 - Array Instance Methods**（略称: **Ruby**）
  - class Array の Instance Methods を 1 日 1 つずつ読んでいく
  - https://docs.ruby-lang.org/en/4.0/Array.html

## Claude Code での使い方

### 基本習慣の記録

```
「今日の記録を開始」
「ラジオ体操とシダキュアは完了」
「スクワット20回完了」
「腕立て15回、腹筋20回やった」
```

### 学習内容の記録

```
「shigの『一貫性の原則』を読んだ。メモ: UIは統一感が大事」
「Rubyの Array#append を読んだ。メモ: 配列の末尾に要素を追加」
```

**略称**: shig (Sociomedia HIG) / Ruby (Ruby 4.0 Array Methods)

## Claude Code の処理フロー

学習内容を記録する際、Claude Code は以下の処理を行います：

### 1. daily/ への記録

- その日の日次ファイル（例: `2026-01-03.md`）に記録
  - 学習項目の方はユーザーが手動入力することもあります
- トピック/メソッド名、URL、メモを記載
- チェックボックスを完了状態にする

### 2. learning/ への詳細ノート作成

#### shig の場合

- ファイル名: `{連番3桁}-{トピック名スラッグ}.md`
- 内容: daily/ からの転記（タイトル、日付、URL、メモ）

#### Ruby の場合

- ファイル名: `{連番3桁}-{メソッド名スラッグ}.md`
- 内容構成:
  1. **基本情報**: メソッドシグネチャと概要
  2. **メモ（daily より）**: daily/ からの転記
  3. **深掘り・補足**: Claude が追加する付加価値情報
     - 重要なポイント
     - ユースケース
     - 注意点
     - 関連メソッドとの比較など

**日本語訳について**: 自動翻訳された日本語に明らかな誤訳がある場合は修正します。

### 3. 連番管理

学習ノートのファイル名は 3 桁の連番（001, 002, ...）で管理します。

- 進捗が分かりやすい
- 同日に複数学習しても対応可能
- ファイルが自然にソートされる
