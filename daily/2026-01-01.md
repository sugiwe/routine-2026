## その他のメモ

```ruby
[0, 1, 2, 3] & [1, 2] # => [1, 2]

# Omits duplicates (重複を除外する):
[0, 1, 1, 0] & [0, 1] # => [0, 1]

# Preserves order from self (自身の順番を保つ):
[0, 1, 2] & [3, 2, 1, 0] # => [0, 1, 2]
[3, 2, 1, 0] & [0, 1, 2] # => [2, 1, 0] 左辺側の順番になるってことか👀
```
