### Ruby 4.0 Documentation - Array Instance Methods

- [x] 今日のメソッド: `<=>`
- URL: https://docs.ruby-lang.org/en/4.0/Array.html#method-i-3C-3D-3E
- メモ: `self <=> other_array → -1, 0, or 1`、通称「宇宙船演算子」

#### コード例・補足

```ruby
# Returns -1, 0, or 1 as self is determined to be less than, equal to, or greater than other_array.
# Iterates over each index i in (0...self.size):
# selfがother_arrayより小さい、等しい、大きいと判断される場合、それぞれ-1、0、1を返す。
# 各インデックスiについて(0...self.size)を反復処理する：

# Computes result[i] as self[i] <=> other_array[i].
# Immediately returns 1 if result[i] is 1:
# result[i]をself[i] <=> other_array[i]として計算する。
# result[i] が 1 の場合、直ちに 1 を返す:
[0, 1, 2] <=> [0, 0, 2] # => 1

# Immediately returns -1 if result[i] is -1:
# result[i] が -1 の場合、直ちに -1 を返す:
[0, 1, 2] <=> [0, 2, 2] # => -1

# Continues if result[i] is 0.
# result[i] が 0 の場合、処理を継続します。

# When every result is 0, returns self.size <=> other_array.size (see Integer#<=>):
# すべての結果が 0 の場合、self.size <=> other_array.size を返します（Integer#<=> を参照）。
[0, 1, 2] <=> [0, 1]        # => 1
[0, 1, 2] <=> [0, 1, 2]     # => 0
[0, 1, 2] <=> [0, 1, 2, 3]  # => -1

# Note that when other_array is larger than self, its trailing elements do not affect the result:
# other_array が self より大きい場合、末尾の要素は結果に影響しないことに注意:
[0, 1, 2] <=> [0, 1, 2, -3] # => -1
[0, 1, 2] <=> [0, 1, 2, 0]  # => -1
[0, 1, 2] <=> [0, 1, 2, 3]  # => -1
```
