### Sociomedia ヒューマンインターフェース ガイドライン

- [x] 今日のトピック: 16. フィッツの法則(Fitts’ Law)
- URL: https://www.sociomedia.co.jp/9146
- メモ: パソコンのスクリーンなどで、どの位置が指し示しやすいかの順番を決める法則。「距離」と「アクセスしやすさ」の両方が関わってくる。例えば四隅は「そこまで行けばマウスが止まる」ので、マウス操作においてはかなりアクセスしやすいポイント。タッチデバイスの場合は四隅のアクセスしやすい性質が無くなるので、順位が変わることになる。
- 感想: 初めて聞く名前の法則だけど、面白い

### Ruby 4.0 Documentation - Array Instance Methods

- [x] 今日のメソッド: `bsearch`
- URL: https://docs.ruby-lang.org/en/4.0/Array.html#method-i-bsearch
- メモ:

```ruby
bsearch {|element| ... } → found_element or nil
bsearch → new_enumerator
```

#### コード例・補足

```ruby
# Returns the element from self found by a binary search, or nil if the search found no suitable element.
# 二分探索で見つかった要素、または適切な要素が見つからなかった場合はnilを返します。
ary = [0, 4, 7, 10, 12]
ary.bsearch {|x| x >=   4 } # => 4
ary.bsearch {|x| x >=   6 } # => 7
ary.bsearch {|x| x >=  -1 } # => 0
ary.bsearch {|x| x >= 100 } # => nil

# See Binary Searching.

# Related: see Methods for Fetching.
```
