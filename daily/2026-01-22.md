#### コード例・補足

```ruby
# Returns a new array containing only the non-nil elements from self; element order is preserved:
# nilでない要素のみを含む新しい配列を返します。要素の順序は保持されます。
a = [nil, 0, nil, false, nil, '', nil, [], nil, {}]
a.compact # => [0, false, "", [], {}]
```
