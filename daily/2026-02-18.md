#### コード例・補足

```ruby
# Returns whether for some element `element` in `self`, `object == element`:
# `self`内のいずれかの要素`element`が`object`と等しい場合に`true`を返します。
[0, 1, 2].include?(2)   # => true
[0, 1, 2].include?(2.0) # => true
[0, 1, 2].include?(2.1) # => false
```
