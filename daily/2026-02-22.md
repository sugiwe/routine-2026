#### コード例・補足

```ruby

# Returns the new string formed by calling method inspect on each array element:
# 配列の各要素に対してinspectメソッドを呼び出して生成される、新しい文字列を返します。
a = [:foo, 'bar', 2]
a.inspect # => "[:foo, \"bar\", 2]"

# Related: see Methods for Converting.
# Also aliased as: to_s
```
