#### コード例・補足

```ruby
# Returns true if exactly one element of self meets a given criterion.
# 自身の要素のうち、指定された条件を満たすものがちょうど1つ存在する場合にtrueを返します。
# With no block given and no argument, returns true if self has exactly one truthy element, false otherwise:
# ブロックも引数も指定しない場合、自身がただ1つの真値要素を持つなら真を返し、それ以外なら偽を返します。
[nil, 0].one? # => true
[0, 0].one? # => false
[nil, nil].one? # => false
[].one? # => false

# With a block given, calls the block with each element in self; returns true if the block a truthy value for exactly one element, false otherwise:
# ブロックが与えられた場合、自身の各要素を引数としてブロックを呼び出します。ブロックが真の値を返した要素がちょうど1つだけであれば true を返し、それ以外の場合は false を返します。
[0, 1, 2].one? {|element| element > 0 } # => false
[0, 1, 2].one? {|element| element > 1 } # => true
[0, 1, 2].one? {|element| element > 2 } # => false

# With argument object given, returns true if for exactly one element element, object === element; false otherwise:
# 引数`object`が与えられた場合、要素`element`のうちただ1つだけが`object === element`を満たしていれば`true`を返し、それ以外の場合は`false`を返します。
[0, 1, 2].one?(0) # => true
[0, 0, 1].one?(0) # => false
[1, 1, 2].one?(0) # => false
['food', 'drink'].one?(/bar/) # => false
['food', 'drink'].one?(/foo/) # => true
[].one?(/foo/) # => false
```
