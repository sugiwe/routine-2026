# Replaces the elements of self with the elements of other_array, which must be an array-convertible object; returns self:
# `self`の要素を`other_array`の要素に置き換えます。`other_array`は配列に変換可能なオブジェクトである必要があります。`self`を返します。
a = ['a', 'b', 'c']   # => ["a", "b", "c"]
a.replace(['d', 'e']) # => ["d", "e"]
