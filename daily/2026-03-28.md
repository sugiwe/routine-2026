#### コード例・補足

```ruby
# Returns elements from self; does not modify self.
# 自身から要素を返します。自身を変更することはありません。

# In brief:
# 概要:
# ※エイリアス`[]`の方が一般的のため例は`[]`を多めに使っていますが、全て`slice`で書き換え可能

a = [:foo, 'bar', 2]

# Single argument index: returns one element.
# 単一のインデックス引数: 1つの要素を返します。
a[0]        # => :foo       # Zero-based index.
a.slice(0)  # => :foo       # 同じ結果
a[-1]       # => 2          # Negative index counts backwards from end.
                            # 負のインデックスは末尾から逆順にカウントします。

# Arguments start and length: returns an array.
# 引数 start と length: 配列を返します。
a[1, 2]        # => ["bar", 2]
a.slice(1, 2)  # => ["bar", 2]  # 同じ結果
a[-2, 2]       # => ["bar", 2]  # Negative start counts backwards from end.
                                # 負の start は末尾から逆順にカウントします。

# Single argument range: returns an array.
# 単一の Range 引数: 配列を返します。
a[0..1]        # => [:foo, "bar"]
a.slice(0..1)  # => [:foo, "bar"]  # 同じ結果
a[0..-2]       # => [:foo, "bar"]  # Negative range-begin counts backwards from end.
                                   # 負の range-begin は末尾から逆順にカウントします。
a[-2..2]       # => ["bar", 2]     # Negative range-end counts backwards from end.
                                   # 負の range-end は末尾から逆順にカウントします。


# When a single integer argument index is given, returns the element at offset index:
# 単一の整数引数 index が与えられた場合、オフセット index の要素を返します:

a = [:foo, 'bar', 2]
a[0] # => :foo
a[2] # => 2
a # => [:foo, "bar", 2]

# If index is negative, counts backwards from the end of self:
# index が負の場合、自身の末尾から逆順にカウントします:

a = [:foo, 'bar', 2]
a[-1] # => 2
a[-2] # => "bar"

# If index is out of range, returns nil.
# index が範囲外の場合、nil を返します。

# When two Integer arguments start and length are given, returns a new array of size length containing successive elements beginning at offset start:
# 2つの整数引数 start と length が与えられた場合、オフセット start から始まる連続した要素を含む、サイズ length の新しい配列を返します:

a = [:foo, 'bar', 2]
a[0, 2] # => [:foo, "bar"]
a[1, 2] # => ["bar", 2]

# If start + length is greater than self.length, returns all elements from offset start to the end:
# start + length が self.length より大きい場合、オフセット start から末尾までのすべての要素を返します:

a = [:foo, 'bar', 2]
a[0, 4] # => [:foo, "bar", 2]
a[1, 3] # => ["bar", 2]
a[2, 2] # => [2]

# If start == self.size and length >= 0, returns a new empty array.
# start == self.size かつ length >= 0 の場合、新しい空の配列を返します。

# If length is negative, returns nil.
# length が負の場合、nil を返します。

# When a single Range argument range is given, treats range.min as start above and range.size as length above:
# 単一の Range 引数 range が与えられた場合、range.min を上記の start として、range.size を上記の length として扱います:

a = [:foo, 'bar', 2]
a[0..1] # => [:foo, "bar"]
a[1..2] # => ["bar", 2]

# Special case: If range.start == a.size, returns a new empty array.
# 特殊ケース: range.start == a.size の場合、新しい空の配列を返します。

# If range.end is negative, calculates the end index from the end:
# range.end が負の場合、末尾から終了インデックスを計算します:

a = [:foo, 'bar', 2]
a[0..-1] # => [:foo, "bar", 2]
a[0..-2] # => [:foo, "bar"]
a[0..-3] # => [:foo]

# If range.start is negative, calculates the start index from the end:
# range.start が負の場合、末尾から開始インデックスを計算します:

a = [:foo, 'bar', 2]
a[-1..2] # => [2]
a[-2..2] # => ["bar", 2]
a[-3..2] # => [:foo, "bar", 2]

# If range.start is larger than the array size, returns nil.
# range.start が配列のサイズより大きい場合、nil を返します。

a = [:foo, 'bar', 2]
a[4..1] # => nil
a[4..0] # => nil
a[4..-1] # => nil

# When a single Enumerator::ArithmeticSequence argument aseq is given, returns an array of elements corresponding to the indexes produced by the sequence.
# 単一の Enumerator::ArithmeticSequence 引数 aseq が与えられた場合、そのシーケンスによって生成されたインデックスに対応する要素の配列を返します。

a = ['--', 'data1', '--', 'data2', '--', 'data3']
a[(1..).step(2)] # => ["data1", "data2", "data3"]

# Unlike slicing with range, if the start or the end of the arithmetic sequence is larger than array size, throws RangeError.
# Range によるスライスとは異なり、算術シーケンスの開始または終了が配列のサイズより大きい場合、RangeError を発生させます。

a = ['--', 'data1', '--', 'data2', '--', 'data3']
a[(1..11).step(2)]
# RangeError (((1..11).step(2)) out of range)
a[(7..).step(2)]
# RangeError (((7..).step(2)) out of range)

# If given a single argument, and its type is not one of the listed, tries to convert it to Integer, and raises if it is impossible:
# 単一の引数が与えられ、その型がリストにないものである場合、Integer への変換を試み、不可能な場合は例外を発生させます:

a = [:foo, 'bar', 2]
# Raises TypeError (no implicit conversion of Symbol into Integer):
# TypeError を発生させます (Symbol から Integer への暗黙的な変換はありません):
a[:foo]

# Related: see Methods for Fetching.
# 関連: Methods for Fetching を参照してください。
# Alias for: []
# [] のエイリアスです
```
