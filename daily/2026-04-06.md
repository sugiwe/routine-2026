#### コード例・補足

```ruby
# Returns a new array that is self as a transposed matrix:
# 自身を転置行列とした新しい配列を返します。
a = [[:a0, :a1], [:b0, :b1], [:c0, :c1]]
a.transpose # => [[:a0, :b0, :c0], [:a1, :b1, :c1]]

# The elements of self must all be the same size.
# Related: see Methods for Converting.
```
