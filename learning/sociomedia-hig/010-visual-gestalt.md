# 視覚ゲシュタルト (Visual Gestalt)

- 日付: 2026-01-10
- URL: https://www.sociomedia.co.jp/9099

## メモ（daily より）

人の知覚は、対象要素を個別に認識するのではなく全体的な枠組みから規定する。この特性を利用したのがプレグナンツの法則（近接、類似、閉鎖）

## 深掘り・補足

### 視覚ゲシュタルトとは

ゲシュタルト心理学における「全体は部分の総和以上である」という原理を、視覚デザインに応用したものです。ユーザーは UI の個々の要素を独立して認識するのではなく、要素間の関係性や配置から全体的なパターンを認識します。

### プレグナンツの法則

人間の知覚は、できるだけシンプルで規則的なパターンを見出そうとする傾向があります。この特性を活用する主な法則：

#### 1. **近接の法則 (Proximity)**

- 物理的に近い要素は、同じグループとして認識される
- **UI での活用例**：
  - フォームのラベルと入力欄を近づけて配置
  - 関連するボタンをグループ化
  - ナビゲーションメニューのセクション分け
- **注意点**：無関係な要素を近づけすぎると、誤った関連性を示してしまう

#### 2. **類似の法則 (Similarity)**

- 色、形、サイズなどが類似した要素は、同じカテゴリーとして認識される
- **UI での活用例**：
  - 同じ色で統一したアクションボタン（例：すべての削除ボタンを赤に）
  - アイコンのスタイルを統一してツールバーを構成
  - 同じフォントスタイルで見出しレベルを表現
- **良い例**：重要度が同じ情報を同じ視覚的重みで表示
- **悪い例**：異なる機能のボタンを同じスタイルにしてしまう

#### 3. **閉鎖の法則 (Closure)**

- 不完全な形でも、人間の知覚は欠けた部分を補完して完全な形として認識する
- **UI での活用例**：
  - 囲み線が完全に閉じていなくても、グループとして認識される
  - 破線や点線でも境界として機能する
  - 背景色の変化だけでセクションを区切る
- **メリット**：完全な線を使わずに視覚的ノイズを減らせる

### その他の重要なゲシュタルト法則

#### **連続の法則 (Continuity)**

- 要素は連続したパターンとして知覚される
- **活用例**：フローチャート、ステップインジケーター、タイムライン UI

#### **図と地の法則 (Figure-Ground)**

- 視覚情報は「前景（図）」と「背景（地）」に分離して認識される
- **活用例**：モーダルダイアログ（前面のダイアログと暗くなった背景）

### 実践的なデザインガイドライン

1. **情報のグルーピング**
   - 関連する情報は視覚的にグループ化する
   - 余白（ホワイトスペース）を効果的に使って区切りを作る

2. **一貫性の維持**
   - 同じ機能には同じ視覚パターンを適用
   - デザインシステムで視覚的ルールを統一

3. **認知負荷の軽減**
   - ユーザーが自然に構造を理解できる配置を心がける
   - 過度な装飾を避け、シンプルなパターンを優先

4. **階層構造の明確化**
   - サイズ、色、配置で情報の重要度を表現
   - 視覚的な重みづけで読む順序を誘導

### まとめ

視覚ゲシュタルトの原理を理解し活用することで、ユーザーが直感的に情報構造を把握できる UI を設計できます。個々の要素をデザインするだけでなく、要素間の関係性を意識した配置が重要です。
