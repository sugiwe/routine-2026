# 68. アイコンのモチーフは特定の文化に依存させない (Culturally Independent Icon Motifs)

- 日付: 2026-03-09
- URL: https://www.sociomedia.co.jp/9736
- daily 記録: [daily/2026-03-09.md](../../daily/2026-03-09.md)

## ユーザーの第一印象（daily より）

これむずいな、歯車とかフロッピーディスクとかはアリなのか？

## 深掘り・補足

### 核心となる原則

国際的なサービスでアイコンを設計する際は、**特定の文化圏でのみ使われるサインや言語依存の慣用表現などを元にした記号を使わない**ことが重要です。

### 問題のある例

記事では 3 つの具体的な例が示されています：

1. **停止標識（八角形の赤い標識）**
   - 多くの国で「止まれ」を意味するが、すべての国で共通ではない
   - 地域によって標識の形状や色が異なる

2. **郵便ポスト（青い箱）**
   - 日本では青い郵便ポストは一般的ではない
   - 国によって郵便ポストの色や形状は大きく異なる（赤、黄色、緑など）

3. **ロケット（アプリ起動のアイコン）**
   - 英語では "launch"（発射・起動）という言葉が共通だが、すべての言語でこの比喩が成立するわけではない
   - 言語的ニュアンスの違いにより、直感的に理解されない可能性がある

### 設計時の注意点

#### ユーザーの質問：歯車やフロッピーディスクは？

- **歯車（設定）**: ソフトウェア業界では広く定着しており、比較的安全
  - ただし、技術に不慣れなユーザーには理解されない可能性もある

- **フロッピーディスク（保存）**: スキューモーフィズムの典型例
  - 若い世代はフロッピーディスクを実物として見たことがない可能性が高い
  - 文化依存というより「時代依存」の問題
  - 現在は慣習として定着しているが、長期的には見直しが必要かもしれない

### 実践的なガイドライン

1. **普遍性の検証**
   - デザインしたアイコンを異なる文化圏の人に見せてテストする
   - 言語や地域に依存しない視覚的メタファーを選ぶ

2. **慣用表現への依存を避ける**
   - 特定の言語の言葉遊びや比喩に基づくアイコンは避ける
   - 例: 英語の "launch" のような言語固有の意味の重なりに頼らない

3. **文化的中立性を保つ**
   - 特定の国や地域の標識、シンボル、色の意味に依存しない
   - 宗教的・政治的なシンボルを避ける

4. **代替案の検討**
   - テキストラベルを併用する
   - より直接的で具体的なビジュアル表現を使う
   - ユーザーがカスタマイズできるようにする
