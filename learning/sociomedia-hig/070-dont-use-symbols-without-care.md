# 70. ◯（マル）✕（バツ）△（サンカク）等の記号を安易に使わない (Don't Use ◯ / ✕ / △ Symbols Without Care)

- 日付: 2026-03-11
- URL: https://www.sociomedia.co.jp/9747
- daily 記録: [daily/2026-03-11.md](../../daily/2026-03-11.md)

## ユーザーの第一印象（daily より）

バツが肯定的なニュアンスで使われることがあるというのは驚き、気をつけようと思った

## 深掘り・補足

### 文化によって異なる記号の意味

日本では「◯=良い、✕=悪い、△=中程度」という使い方が一般的ですが、これは**日本特有の文化的慣習**です。国際的なインターフェースでは通用しません。

**特に✕記号の問題:**
- 日本: 否定的な意味（不正解、NG、ダメ）
- 欧米圏: **チェックマークと同じ肯定的な意味**で使われることがある
- つまり、同じ✕記号でも真逆の意味になる可能性があります

### 具体例: PlayStation コントローラー

PlayStation のコントローラーは、この問題を象徴する実例です:

- **日本版**: ◯ボタンで決定、✕ボタンでキャンセル
- **欧米版**: ✕ボタンで決定、◯ボタンでキャンセル

同じハードウェアでも、地域によってボタンの意味が逆転しています。これは文化的な記号認識の違いを明確に示しています。

### 国際的なデファクトスタンダード

近年の傾向として:
- **✓（チェックマーク）と✕（バツ）** の組み合わせが国際的に普及しつつあります
- ✓ = 肯定、選択、完了
- ✕ = 否定、未選択、削除

ただし、これも完全に統一されているわけではないため、注意が必要です。

### 推奨される対応

1. **文字による明示を優先**
   - 「OK」「キャンセル」「はい」「いいえ」など、言語で明確に表現
   - 記号だけに頼らない

2. **国際的に認知されたアイコンを使用**
   - ✓（チェック）と✕を使う場合は、文脈を明確に
   - 色も併用（緑=OK、赤=NG など、ただし色覚多様性にも配慮）

3. **ターゲットユーザーの文化を考慮**
   - 多言語対応の場合は、記号の意味が文化間で異ならないか検証
   - ローカライズ時に記号の使い方も見直す

4. **テストとフィードバック**
   - 異なる文化圏のユーザーでユーザビリティテストを実施
   - 記号の意味が正しく伝わっているか確認

### まとめ

「◯✕△」は日本独自の記号体系であり、グローバルなインターフェースでは誤解を招く可能性があります。記号だけに頼らず、テキストやユニバーサルに理解されるアイコンと組み合わせることで、文化を超えた明確なコミュニケーションを実現できます。
