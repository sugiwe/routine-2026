## 💡 学習の進め方

1. **daily/ にメモを残す**
   - 「今日読んだ部分」と「メモ」を記録
   - 気軽に、気づいたことをメモする

2. **詳細を残したいとき**
   - `learning/writebook/YYYY-MM-DD-{トピック}.md` を作成
   - daily からリンクを貼る
   - このREADME にもリンクを追加

3. **無理しない**
   - 毎日詳細ノートを作る必要はない
   - daily のメモだけでもOK
   - コツコツと、無理せず続ける

## 🎯 学習の重点ポイント

- コードの構造と設計パターン
- Hotwire（Turbo / Stimulus）の実装
- Rails のベストプラクティス
- テストの書き方
- パフォーマンスの考慮事項

---

**無理せず、コツコツと。それが一番続くコツです 💪**
